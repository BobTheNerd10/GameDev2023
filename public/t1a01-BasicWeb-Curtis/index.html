Scrolling text :D

This scrolling text here is a marquee! Pretty neat, huh? They've mostly fallen out of fashion in web development now because of how annoying they look, but they used to be a much bigger part of the web. It's like a news ticker! You can also put images in them to make them look EXTRA annoying! (but i've spared you of that because it's incredibly distracting)


One last thing to note of is <div>s! Divs let you divide up your webpage and add lots of cool CSS and JS to them! As useful as divs are, you can use most HTML tags to serve the same purpose (this page is divided up into <header> and <main>, which all do the exact same things as div, but they make your code a lot more readable and are good practice! Typically you'll also see <footer>, but that's not used on this web page because this page has no footer.) They're very common in web page development, so it's worth mentioning them as one of the basic elements as the web!

The divider line below this bit of text is called an <hr>! They're pretty nifty for making your web page's content look nice :D
